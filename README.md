# 📚 Study Assistant CLI

> A modular Python command-line application for organizing study notes, searching information, saving important answers, and exporting summaries. This project is being developed incrementally into a complete AI-powered study assistant using FastAPI, OpenAI, Retrieval-Augmented Generation (RAG), Vector Databases, and Flutter.

---

## 🎯 Project Overview

Students often keep their lecture notes, assignments, and study material scattered across multiple files and folders, making it difficult to quickly locate information while studying.

Study Assistant CLI solves this problem by providing a centralized command-line application where users can:

- Organize notes by subject
- Search study material
- Save important answers
- Export summaries
- Build a structured personal knowledge base

Rather than creating multiple unrelated projects, this repository follows a continuous development approach where the same application evolves into a production-ready AI system.
